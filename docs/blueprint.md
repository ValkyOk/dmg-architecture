# **App Name**: DMG Architecture Hub

## Core Features:

- Project Portfolio: Showcase architectural projects with images, descriptions, and filtering by type (plans, renders, completed projects).
- Service Quotation: Allow users to request quotes for architectural services by specifying project details and requirements.
- Client Registration: Allow new clients to sign up.
- Professional Registration and Access: Enable architects and professionals to register, login, and access a private area for uploading/downloading project resources.
- Multi-Language Support: Provide content in both Spanish and English, with an easy language selection option available on every page.

## Style Guidelines:

- Primary color: Deep olive (#424c19) to reflect sophistication and a connection to the earth.
- Background color: Very light gray (#f0f0f0) for a clean and modern aesthetic.
- Accent color: Pale olive (#98ab6f) for subtle highlights and interactive elements.
- Font pairing: 'Belleza' (sans-serif) for headlines to bring out the artistic and fashionable touch and 'Alegreya' (serif) for body text, lending elegance and readability.
- Responsive layout: Ensure the website adapts seamlessly to different screen sizes, maintaining usability on desktops, tablets, and mobile devices.
- Minimalist icons: Use clean and simple icons to represent different services and sections, enhancing the overall aesthetic.
